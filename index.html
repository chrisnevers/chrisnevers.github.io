Projects

I'm usually always working on a programming project. Two of my larger projects are a Musical Chord Analyzer and a Racket Compiler

Musical Chord Analyzer

written in Python

  • Scrapes artist/song chord data from online
  • Determines the key of a song
  • Provides roman numeral analysis of songs
  • Generates a visual directed graph of chord changes
  • Generates random chord progressions (of any length) in the style of the selected artist/song
